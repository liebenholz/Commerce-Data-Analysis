# Commerce Data Analysis
Kaggle Data Analysis and Visualization Hands-on Project Using SQL, Tableau, and Python by LIKE LION.

- **SQL**: Discovering Analysis Topics Through Exploratory Data Analysis(EDA)
- **Tableau**: Data Storytelling Through Dashboards and Stories
- **Python**: Uncovering New Insights Through Analysis with SQL and Tableau
- Dataset: [Brazilian E-Commerce Public Dataset by Olist](https://www.kaggle.com/datasets/olistbr/brazilian-ecommerce)
